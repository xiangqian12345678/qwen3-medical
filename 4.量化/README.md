## 显存分析

32B大模型部署显存：

| 部署策略              | 显存需求                     |
  |-------------------|--------------------------|
| FP16/BF16 单卡      | ≥ 80GB（模型 + 激活 + buffer） |
| FP16 + 8K context | 90–100GB                 |
| FP32 单卡           | ≥ 130–140GB              |

    32B量化大模型部署显存：

| 量化类型      | 权重显存    | 激活显存   | 总显存估计   |
  |-----------|---------|--------|---------|
| FP16      | 64GB    | 6–12GB | 70–80GB |
| INT8      | 32GB    | 6–12GB | 38–44GB |
| INT4      | 16GB    | 6–12GB | 22–28GB |
| INT4+GPTQ | 10–11GB | 6–12GB | 16–23GB |
